日本北九州18天背包旅遊行程計劃

這次北九州18天的背包旅遊, 經過多次刪減, 最後決定只安排了遊覽北九州五個縣 – 福岡縣、大分縣、熊本縣、長崎縣和佐賀縣。盡管有朋友多次說這路線只需五天便可完成, 我還是堅決不變。在出發前兩星期, 更取消了往阿蘇火山、刪減了長崎縣兩個景點 – 佐世保和平戸。這樣, 北九州每個主要城市便可停留二至三天, 十分輕鬆。而最後證實這安排是絕對正確的。

北九州  JR PASS

為了省錢, 只購買了一張北九州 5天 JR PASS, 費用是每張港幣565.00 。在第六天才開始使用, 主要利用這 JR PASS 從北九州的小倉移動到熊本, 相信是整個旅程交通費最昂貴的一段行程。利用北九州 5天 JR PASS 的結果如下:

  • [普通]日田彦山線: 小倉 → 日田
  • [特急]由布院之森5号: 日田 → 由布院
  • [普通]JR 久大本線: 由布院 → 大分
  • [特急]ソニック18号: 大分 → 杵築
  • [普通]日豊本線: 杵築 → 別府
  • [普通]日豊本線: 別府 → 大分
  • [普通]豊肥本線: 大分 → 緒方
  • [普通]豊肥本線: 緒方 → 大分
  • [特急]九州横断特急1号: 大分 → 熊本
  • [普通]鹿児島本線: 熊本 → 荒尾
  • [快速]くまもとライナー: 荒尾 → 大牟田
  • 新幹線: 久留米 → 熊本

因從大牟田乘西鐵往柳川是臨時決定的, 出發前也沒有準備任何資料, 以為 JR 駅是在西鐵久留米站附近的, 結果走了差不多一小時才到達, 但卻認識了一位背著結打的日本年輕歌手, 沿途與他合唱了幾首 BEATLES 的名曲, 十分開心, 所以不覺路途遙遠。

JR 劃位

為了行程更富彈性, 在到達福岡當天只於座位比較緊張的兩條路線預先劃位:

  • [特急]由布院之森5号: 日田 → 由布院
  • [特急]九州横断特急1号: 大分 → 熊本

由布院之森比想像中的爆滿。在3月16日傍晚抵福岡, 計劃是在她生日3月21日那天搭乘由布院之森往由布院。滿以為預定6天後的座位, 應該沒有問題。但當在博多JR購票大堂門前看見劃位通告, 3/21的由布院之森竟然全部滿座, 不禁嚇了一跳。

幸好我們選擇在日田搭乘由布院之森5号往由布院, 剛好有兩位乘客在日田下車。結果當天她還在車上得到由布院之森贈送了一份禮物。/p>

天氣

天氣的好與壞足以影響整個行程。出發前參考了 Accuweather.com 網站的天氣預測, 下雨、多雲及陰天竟然佔去16天, 而真正的晴天只有2天! 當時真的害怕沒法按原定計劃完成行程。

最後結果只有兩天在博多下著大雨, 令部份行程更改, 其餘的日子都是晴天和陰天, 大部份行程得以順利完成。
