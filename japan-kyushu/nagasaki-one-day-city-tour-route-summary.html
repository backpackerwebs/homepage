長崎市豐富緊湊的一天行程大綱

3.28.2015 (星期六)

昨天因交通的延誤, 差不多下午四時才到達長崎駅, 結果取消了原定下午遊覽長崎東山手區、南山手區和出島附近一帶的景點。為了追回昨天下午沒去的部份景點, 今天的行程差不多將長崎原本安排一天半的行程濃縮為一天, 結果從清早走到晚上, 真是十分豐富緊湊, 也十分疲累。

以下是今天的行程大綱:

上午:
電車: 長崎駅→ 浜口町站
步行: 國立長崎原爆死沒者追悼平和祈念館 → 長崎原爆資料館 →
原爆公園 → 松山町防空洞群遺跡 → 平和公園
電車: 松山町→ 出島

午餐: 出島碼頭海鮮刺身

下午:
步行: 出島碼頭 → 県庁前
巴士: 県庁前站 → 風頭山
步行: 風頭山巴士總站 → 風頭公園 → 龍馬通り → 若宮稻荷神社 →
龍馬足靴像 → 龜山社中の跡 → 龍馬通り → 深崇寺 →
眼鏡橋 → 賑橋 → 青年会館跡
電車: 賑橋 → 思案橋
步行: 參觀孫文先生故緣之地
電車: 思案橋 → 築町
步行: 築町 → 長崎新地中華街 → 福建通り → 長崎龍馬の道 →
荷蘭大道 → 荷蘭坡 → 荷蘭坡大道 → 荷蘭坡 → 孔子廟 →
舊香港上海銀行長崎支店紀念館 → 大浦天主堂 →
国際あいさつ通り → 哥拉巴天道

傍晚:
步行: 哥拉巴天道 → 鍋冠山公園展望台 → 哥拉巴天道觀景台
電車: 石橋 → 築町 → 大波止

今天的行程十分精彩, 也十分順利。

風頭公園

滿山櫻花怒放的長崎風頭公園, 一片粉紅色的世界, 風景美得無法形容。

長崎 龍馬通り

沿蜿蜒於山中的龍馬通り從風頭山頂走到長嶇市區的深崇寺, 是一段令人難忘的歷史郊遊遠足路線。

長崎 荷蘭坡(オランダ坂)

我們從荷蘭大道的荷蘭坡經荷蘭坡大道(オランダ坂通り)走到石橋, 把頭尾兩段的荷蘭坡及整條荷蘭坡大道徹底的走一遍。沿途沒有其他遊客, 十分寧靜。

長崎 哥拉巴天道 展望台

站在哥拉巴天道展望台, 長崎市就在腳下。

長崎 鍋冠山公園展望台

從哥拉巴公園第2入口出發,向上爬了差不多半小時才到達鍋冠山的山頂, 比估計的高很多, 也比稲佐山遊步道辛苦很多。登上鍋冠山展望台, 無盡景色在腳下, 總算不枉此行。

長崎 哥拉巴天道觀景台夜景

其實哥拉巴天道觀景台的夜景毫不遜色, 而且安全很多。

總括今天的行程, 可以用精彩、豐富和難忘來形容。
